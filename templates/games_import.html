Import PGN

Paste one or more games in PGN format (OTB games, exports from other sites), or upload a .pgn file. Duplicates are detected automatically, players are matched to existing ones by name, and imported games show up under the manual source filter.

Notes: chess variants and games from custom start positions are skipped. Games without moves or without player names are skipped. If a player name matches one of your existing players (any account name, or a display name like a broadcast GM), the game is attached to that player — use flask link-identity to merge identities later if needed.
